ℹ️ Prérequis

Les Étapes Détaillées

1

1. Préparation du dossier

Créez un dossier dédié pour organiser tous les certificats.

mkdir siem-certs
cd siem-certs
2

2. Création de l’Autorité de Certification (CA)

Cette étape crée la racine de confiance de votre infrastructure.

openssl genrsa -out ca-root.key 4096

openssl req -x509 -new -nodes -key ca-root.key -sha256 -days 3650 \
  -out ca-root.crt -subj "/CN=Smart-SIEM-Root-CA/O=UCAC-ICAM/OU=Promo2028"
3

3. Création du fichier de configuration (v3.ext)

Ce fichier définit les noms et adresses IP autorisés.

authorityKeyIdentifier=keyid,issuer
basicConstraints=CA:FALSE
keyUsage = digitalSignature, keyEncipherment, dataEncipherment
subjectAltName = @alt_names

[alt_names]
DNS.1 = localhost
DNS.2 = siem_serveur_ucac-icam
IP.1 = 127.0.0.1
IP.2 = 192.168.1.100
IP.3 = 192.168.1.150
4

4. Génération du certificat du serveur

openssl genrsa -out siem-server.key 2048

openssl req -new -key siem-server.key -out siem-server.csr -subj "/CN=siem_serveur_ucac-icam"

openssl x509 -req -in siem-server.csr -CA ca-root.crt -CAkey ca-root.key \
  -CAcreateserial -out siem-server.crt -days 730 -sha256 -extfile v3.ext

Déploiement des fichiers

Smart Agents

Copiez ca-root.crt dans le dossier Agents/agent/certs/

Serveur Smart SIEM

Copiez siem-server.crt et siem-server.key dans le dossier de votre application.
