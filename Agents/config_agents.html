Le Smart SIEM Agent est un composant logiciel de sécurité léger et autonome conçu pour s'exécuter en arrière-plan avec des privilèges élevés (root / SYSTEM). Son rôle est de collecter en temps réel les données de télémétrie d'un point de terminaison (endpoint), de les structurer, et de les transmettre de manière sécurisée à un serveur SIEM centralisé pour analyse et corrélation.

I Fonctionnalités de l'Agent

L'agent exécute plusieurs tâches modulaires de surveillance et de collecte de données, orchestrées par un ordonnanceur interne (Scheduler). Chaque module est conçu pour être cross-platform (compatible Windows et Linux via des abstractions système) :

1. Collecte de Métriques de Performance
  • Moniteur CPU : analyse la charge globale de calcul du processeur en temps réel pour détecter des pics anormaux.
  • Moniteur Mémoire (RAM) : surveille le taux d'utilisation de la mémoire physique et détecte les fuites ou les allocations massives suspectes.
2. Surveillance du Stockage et du Système de Fichiers
  • Filesystem & Partitions : recense l'espace disque disponible, les points de montage (Linux) et les lettres de lecteurs (Windows).
3. Télémétrie Réseau & Connexions
  • Network Collector : énumère et analyse l'intégralité des connexions réseau actives (sockets TCP/UDP, adresses IP sources/destinations, ports ouverts).
4. Inventaire des Processus et Services
  • Process Monitor : liste l'activité des processus en cours d'exécution, leurs identifiants (PID) et les ressources consommées.
  • Services Monitor : assure le suivi de l'état des services système (Systemd sous Linux, Gestionnaire de services sous Windows).
5. Collecte et Centralisation des Événements (Logs)
  • Logs Collector : récupère les journaux d'authentification et de sécurité du système.
  • Transfer Manager : compresse, structure au format JSON et synchronise les événements collectés.
6. Canal de Communication Bidirectionnel (SOAR & Réponses Actives)
  • Command Service (WebSocket App) : maintient une connexion persistante et bidirectionnelle avec le serveur SIEM.

II Installation & Configuration Linux · Ubuntu

L'agent Linux est packagé sous la forme d'un fichier binaire unique au format ELF (généré via PyInstaller).

1. Prérequis Système

  • Système d'exploitation : Ubuntu 20.04 LTS / 22.04 LTS / 24.04 LTS (x86_64).
  • Dépendance Graphique : pour le premier lancement visuel uniquement, l'environnement graphique X11/Wayland doit disposer du paquet Tkinter :
sudo apt update && sudo apt install -y python3-tk

2. Déploiement et Premier Lancement

L'exécutable intègre un mécanisme d'élévation graphique via pkexec permettant le double-clic sécurisé.

  1. Déposez le fichier binaire compilé smart-siem-agent sur la machine.
  2. Lancez-le sans sudo : 
    ./smart-siem-agent
  3. Une fenêtre d'authentification native Ubuntu surgit pour demander le mot de passe administrateur.
  4. L'Assistant Graphique Moderne Smart SIEM apparaît à l'écran.
  5. Renseignez les informations de connectivité réseau de votre infrastructure.
  6. Cliquez sur « Installer & Démarrer l'agent ».

3. Sous le capot Linux

L'assistant met à jour la configuration, copie le binaire dans /usr/local/bin, crée le service systemd et l'active.

sudo systemctl daemon-reload
sudo systemctl enable smart-siem.service
sudo systemctl start smart-siem.service

4. Commandes d'Administration

  • Statut : sudo systemctl status smart-siem.service
  • Logs : sudo journalctl -u smart-siem.service -f
  • Arrêt : sudo systemctl stop smart-siem.service
  • Désinstallation : 
    sudo systemctl stop smart-siem.service
sudo systemctl disable smart-siem.service
sudo rm -f /etc/systemd/system/smart-siem.service
sudo rm -f /usr/local/bin/smart-siem-agent
sudo systemctl daemon-reload

III Installation & Configuration Windows

L'agent Windows est packagé sous la forme d'un exécutable unique autonome smart-siem-agent.exe.

1. Prérequis

  • Système : Windows 10 / 11 / Server (64-bit).
  • Outil tiers : nssm.exe (Non-Sucking Service Manager) doit être inclus.

2. Déploiement

  1. Transférez smart-siem-agent.exe sur la machine (ex. C:\temp).
  2. Exécutez en tant qu'administrateur.
  3. Remplissez les paramètres réseau dans l'interface.
  4. Cliquez sur « Installer & Démarrer l'agent ».

3. Sous le capot Windows

L'assistant copie les fichiers dans C:\Program Files\SmartSIEM et crée le service SmartSIEMAgent via nssm.

nssm.exe install SmartSIEMAgent "C:\Program Files\SmartSIEM\smart-siem-agent.exe"
nssm.exe set SmartSIEMAgent Start SERVICE_AUTO_START
sc start SmartSIEMAgent

4. Commandes d'Administration

  • Statut : sc query SmartSIEMAgent
  • Arrêt : net stop SmartSIEMAgent
  • Désinstallation : 
    sc stop SmartSIEMAgent
"C:\Program Files\SmartSIEM\nssm.exe" remove SmartSIEMAgent confirm
rmdir /s /q "C:\Program Files\SmartSIEM"
